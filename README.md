# myAquarium

Manage your aquarium:

 * Add fish or jelly fish
 * Add seaweed
 * Choose their propeties
 * Feed them when they are hungry


For this work i use Java 1.8 and GUI.

There are several Design Patterns:

 * Singleton
 * Abstract Factory
 * Prototype
 * Listener/Observer
 * Decorator
 * Memento
 * State
